# Arduino MKR 1010

Ardiono IDE . Programming . MKR IoT Carrier

## Introduction

The Arduino MKR 1010 is a microcontroller board that is designed for makers, educators, and IoT (Internet of Things) developers who want to build connected devices easily. 

In this exercise, you'll learn how to set up your Arduino MKR 1010 board and write a simple program to display the message "Hello, World!" on the Serial Monitor.

## Equipment
+ Arduino MKR 1010 board 
+ Micro USB cable
+ Computer
+ A WiFi network
