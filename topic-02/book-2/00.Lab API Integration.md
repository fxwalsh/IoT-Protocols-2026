# Web API Integration

RPi . Open Weather Map . Python

## Introduction

This lab is an introduction to integrating web-based services with IoT devices. You will use the Raspberry Pi with the OpenWeatherMap API, which provides current and historical weather information. The lab will help your understanding of API usage, Python programming, and data handling in programs, and prepares for more complex labs involving data fetching, processing, and hardware interfacing.

## Equipment
+ Raspberry Pi 3/4
+ microSD Card with Raspberry OS
+ Internet Connection for RPi

## Software
+ Text editor 
+ Laptop/Computer

## Objectives
+ Research an API
+ Write/update a Python Program
+ Programmatically request and process data from an API
