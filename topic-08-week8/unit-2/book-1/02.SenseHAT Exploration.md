### Reading Sensor Data

#### 1. Temperature
   - The Sense HAT has a temperature sensor that can be accessed as follows:
     ```python
     temp = sense.get_temperature()
     print(f"Temperature: {temp} °C")
     ```
     
   - **Explanation**: `.get_temperature()` reads the ambient temperature in Celsius. 

#### 2. Humidity
   - Measure the humidity level in the environment:
     ```python
     humidity = sense.get_humidity()
     print(f"Humidity: {humidity:.2f} %")
     ```
   - **Explanation**: `.get_humidity()` returns the relative humidity as a percentage.
   - **Number Formatting:** The `:.2f` format specifier in the line `print(f"Humidity: {humidity:.2f} %")` is used to format the value of `temp` as a floating-point number with two decimal places. the`:` starts the formatting specification, `.2` limits the number to two decimal places, and `f` specifies that the value should be formatted as a floating-point number.

#### 3. Pressure
   - Access the barometric pressure:
     ```python
     pressure = sense.get_pressure()
     print(f"Pressure: {pressure:.2f} Millibars")
     ```
   - **Explanation**: `.get_pressure()` reads the atmospheric pressure in millibars.

### Orientation and Movement Sensors

#### 4. Gyroscope
   - Measure the orientation of the Sense HAT using the gyroscope:
     ```python
     gyro = sense.get_orientation()
     print(f"Gyroscope - pitch: {gyro['pitch']:.2f}, roll: {gyro['roll']:.2f}, yaw: {gyro['yaw']:.2f}")
     ```
   - **Explanation**: `.get_gyroscope()` gives the pitch, roll, and yaw values in degrees.

#### 5. Accelerometer
   - Read the acceleration in the x, y, and z axes:
     ```python
     accel = sense.get_accelerometer_raw()
     print(f"Accelerometer - x: {accel['x']:.2f}, y: {accel['y']:.2f}, z: {accel['z']:.2f}")
     ```
     
   - **Explanation**: `.get_accelerometer_raw()` returns the raw x, y, and z acceleration values.

     

#### 6. Magnetometer (Compass)
   - Use the magnetometer to get the direction:
     ```python
     mag = sense.get_compass()
     print(f"Compass: {mag:.2f}°")
     ```
   - **Explanation**: `.get_compass()` provides the heading in degrees relative to magnetic north.
