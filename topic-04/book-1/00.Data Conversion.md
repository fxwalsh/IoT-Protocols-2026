# Data Converters

Analog to Digital(ADC) and Digital to Analog(DAC) lab

This experiment will use an ADC data converter on the Raspberry Pi to construct a digital light meter/analog input. The converted input signal will be displayed numerically as a fixed-point number on the terminal, using a integer format.![talk-2](./img/main.png)

## Materials Needed:

1) Raspberry Pi 3
2) Breadboard
4) MCP3008 ADC
5) Jumper wires
6) A potentiometer
7) 1kΩ, 10kΩ resistor
