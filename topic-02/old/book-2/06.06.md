# Carrier Examples and Cheat Sheet

### MKR IoT Carrier Rev2 Cheat Sheet

Go [here](https://docs.arduino.cc/tutorials/mkr-iot-carrier-rev2/cheat-sheet) to learn how to set up and use the MKR IoT Carrier Rev2.

If you scroll down, you will see some example code that you can try out.

### IDE Examples

The Arduino IDE includes example sketches for the MKR Carrier. Go to `File->Examples->Arduino_MKRIoTCarrier` and try a few out.
