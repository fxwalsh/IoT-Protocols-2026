# Simulating IoT in Packet Tracer

The activities in this lab introduce the IoT features included in Packet Tracer v7.1 and later editions.
Please complete the following Packet Tracer Labs taken from Cisco Academy course.

![Smart Home Simulation](./img/Capture.png)

In these exercises you will:

+ Add Devices to a Packet Tracer Smart Home Example
+ Connect to a Packet Tracer Registration Server
+ Modify and Monitor Environment Variables
+ Create and Modify a custom IoT component
+ Program an IoT component
